🍎 FruitBox 테스트 게임

🔧 Chrome 확장 테스트용 페이지
이 페이지는 AlphaApple Chrome 확장을 테스트하기 위한 샘플 게임입니다.
확장을 설치한 후 이 페이지에서 AI Assistant를 테스트해보세요!
점수: 0

📖 게임 규칙

  • 17×10 격자에서 직사각형을 드래그해서 선택
  • 선택한 직사각형 안의 숫자 합이 정확히 10이면 성공
  • 성공하면 해당 영역 제거, 제거한 칸 수만큼 점수 획득
  • 더 이상 제거할 수 없으면 게임 종료

🤖 AI Assistant 사용법

  1. Chrome 확장이 설치되어 있는지 확인
  2. 페이지 로드 후 우하단 "🍎 AI Assistant" 버튼 클릭
  3. AI가 추천하는 직사각형이 금색으로 하이라이트됨
  4. 추천 영역을 클릭하거나 드래그해서 선택
  5. 게임 진행 후 다시 AI 버튼 클릭해서 다음 수 추천 받기
